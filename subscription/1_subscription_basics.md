## Subscription Basics

Install Subscription module

Go to Subscription App -> Products -> Create a product

Product must enable the `Subscription` checkbox

`Recurring Prices` tab will show. Add a recurring plan and price to this tab/table.

Create a pricelist for subscription by going to Products -> Pricelist -> New -> add price to the `Recurring Prices` tab.

Go to Subscription App -> Subscription -> Create a subscription quotation

Select a recurring plan, Payment Terms, and pricelist

Add product on the order line

Select the product subscription created.

confirm the quotation to become an SO.

Create invoice for SO.

in the invoice, you can see the start and end date of the subscription of the user

in the SO page, a new field named `Next Invoice` will show that indicates when is the next invoice will be created and sent to the customer.
