## Alternative Work Center

This allows a work order to be rerouted to an alternative work center in the case that the main work center is busy or blocked for any other reason

Go to Configuration -> Work Centers. Create a new Work Center

Add other work center in the `Alternative Workcenters` field.

Go to Equipment tabs to add equipments to work center

create an MO and use a BoM. If incase a work center is already used by other MO, check the `Plan` button if its highlighted then click it so that the Work Center will change to the alternative work center
