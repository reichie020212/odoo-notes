## Maintenance Requests

To create a Maintenance Requests, Go to Maintenance App -> Maintenance -> Maintenance Requests -> Click `New`

another way to create Maintenance Requests, Go to Maintenance App -> Maintenance -> Maintenance Calendar -> Select date and time then add name, then click create

another way to create Maintenance Requests, Go to Shop Floor App -> Select a Work Center -> Select a MO card then click its gear icon -> a pop-up window will appear -> click Request Maintenance

another way to create maintenance Requests, Got Manufacturing App -> Manufacturing Orderes -> select a confirmed MO -> click the `Maintenance Request` header button

in the Maintenance Requests Form view, fill-up the following field
- For field - to specify if the request is being created for a price of equipment or the entire work center
- Equipment field - will only show if `Equipment` is selected in `For` field.
- Work Center field - will only show if `Work Center` is selected in `For` field.
- Block Workcenter field - will only show if `Work Center` is selected in `For` field. This prevents work orders from being scheduled at the work center.
- Maintenance Type - either `Corrective` or `Preventive`
  - Corrective is intended to fix an existing issue
  - Preventive is intended to prevent issues from occuring in the future

to manage Maintenance Requests, Go to Maintenance App -> Dashbaord -> Check the Maintenance Teams Cards and click the `To Do` button
