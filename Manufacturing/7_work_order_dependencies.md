## Work Order Dependencies

Go to Configuration -> Settings

Enable Work Orders and Work Order Dependencies under it

Go to Bill Of Materials form -> Miscellaneous tab

Check `Operation Dependencies` field.

Go to Operations Tab

Create or update an operation, then select another operation on `Blocked by` field. This means that the selected operation must be finished first.

Create an MO then check the fields, Components, and Work Orders.

Once done checking, click the `Plan` button

Go to Planning -> Planning by Production to see the overall calendar
