## Flexisble Consumption

Go to BoM menu then open or create a BoM

Go to `Miscellaneous` tab.

Set `Flexible Consumption` field based on your preference

if `Flexible Consumption` is set to `Blocked`, it means that only managers will be able to close the MO for which a difference component quantity has been used

For now set it to `Allowed with Warning`

Create an MO then on the Components table, update the quantity of a component, then check the `consumed` column

Once done producing the product or by clicking the `Produce All` header button, a pop-up window will show about Consumption Warning.

This pop-up will show you the components changed, the original quantity, and the current quantity

if `Blocked` is selected, `Confirm` button in the pop-up will not be visible for regular users and only manager will be able to see and click it.

click `Confirm`.
