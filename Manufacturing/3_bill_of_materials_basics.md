## Bill of Materials Basics

Go to Configuration -> Settings -> Enable `Work Orders`

Go to products form -> Inventory tab -> Check the `Manufacture` Routes

Enabling `Manufacture` Route tells odoo that the product can be manufactured which allows us to create MOs for it

Create BoM by going to Products -> Bill of Materials or in product form then `Bill of Materials` smart button

set the product, quantity, bom type, and other fields.

set BoM Type to `Manufacture this product` for now.

in Components tab, add a component and quantity

in Operations tab, Add operations

You can now use the BoM when creating a MO
