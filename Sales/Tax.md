Go to apps menu then search `tax` to see all tax modules

Go to Accounting -> Configuration -> taxes

In tax form, select tax computation
- Group of taxes
- fixed
- percentage
- percentage tax included

Select Tax type - Sales, Purchase, None

Select Tax Scope - Goods, Services

In distribution for invoices table, there should be one line for base. base refers to the order line

and one line for tax percentage

Tax grids is used to generate tax reports automatically

Check advanced options tab to see if there's something that is needed to configure

Go to Configuration -> settings to setup default taxes

Fiscal Position in Accounting -> Configuration -> Fiscal

Fiscal Position refers to the rules and configurations that dictate how taxes and accounts are applied to transactions based on location of the customer

A Fiscal Position can be selected by a res.partner by going to Contact -> Sales and Purchase tab -> Fiscal Position field
