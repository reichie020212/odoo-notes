Enable `Purchase Order Approval` in Purchase Settings

Set Minimum Amount

Go to Manage User form and change the Purchase Access rights of the user to `User`

Enable `Warnings` in Purchase Settings

Go to Purchase -> Orders -> Vendors -> Internal Notes tab -> Add Warning message

Go to Products form -> Purchase Tab -> then add warning message on the warning section

When a vendor or product used in creating RFQ have a warning setup, this warning will show.

When an RFQ with above the minimum amout set and is created by a `User`, the RFQ needs to be approve by a `Administrator` of Purhcase
