## Navigation: Links and Menus

3-click rule: suggest that all information on our website should be available within three mouse clicks.

This rule underscores the importance of organizing our website efficiently with clear menus and relevant buttons.

to add a link to a text, highlight the text then go to customize then click the `Insert and edit link` button

to add an anchor to a block, click the block and then click the `Create a link to target this section` button

Link can be a button by updating the style field into `Custom`.

to customize the menu, click the header menu and go to customize section

you can change the template of the header menu

click each item of the menu to customize them

To add, edit, or delete a menu, go to Site menu -> Menu Editor
