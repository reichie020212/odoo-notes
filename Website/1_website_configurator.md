## Website Configurator

Install website module

There are 2 ways to create a website.

The first way is when you install website the first time on a fresh database, the website configurator will ask you several questions
regarding the website you want. Answer them and website configurator will automatically create one

The second way is by manually creating your own website inside the website module

After creating the website, go to Website -> Configuration -> Settings to modify the settings of website
