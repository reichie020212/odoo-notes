## Translation

to add languages, go to websites -> configurations -> settings -> Add languages to `languages` field

in the webpage, select the language you added

Not all text will be automatically translated to the language, so you have to manually translate them by clicking the edit button then translate

after clicking `translate`, you will notice that the text now have two colors, yellow and green.

Yellow means the text still has to be translated

Green means the text is already translated

Select the yellow text and translate them manually then save.
