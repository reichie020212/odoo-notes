## Packages

Containers used to transfer a bunch of product all at onces

This container can be anything you want like a box or a cart

This is also used in shipping products to our customer and in batch wave and cluster picking

enable `packages` in config setting

in Receipt stock picking, click the `Put in Pack` button. This button wil automatically add the products of stock picking to a package. A `packages` smart button will show

click the hamburger menu to check the destination package

destination package can be change or updated here

click `packages` smart button to see the packages

go to products -> packages to see all packages

go to configuration -> operation type then set the `move entire packages` to True

In inventory overview, create a new internal transfer

a table for package is now available, add the existing packages here

Click `Put in pack` button if you want to pack the product to create a new package

validate stock picking then click `moves` smart button
