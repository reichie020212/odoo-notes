## Custom Returns Operation

Create `Returns` Operation type and set the following fields
- Type of operation: Receipt
- Sequence prefix
- Default source location: partners/customers
- Default Destination location:

Open the `Delivery orders` operation type to set the `returns type` field to the `returns` operation type

in a validated SO stock picking, a `Return` header button is now showing

by clicking the button, a `Reverse Transfer` pop-up window will show

set the product and quantity

set the return location

a stock picking for return will be automatically created and open

click `Quality Checks` then determine if it pass or fail with quality check notes

validate the return order
