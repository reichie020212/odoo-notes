## Cluster Picking

is an advanced method for fulfilling multiple orders at once by having a single picker gather products for several orders into dedicated containers simultaneously

Enable `packages` and `Batch, wave, and cluster transfer`

Go to warehouse and set two or three step outgoing shipment

Create Packages in Products -> Packages to be used by cluster picking

Set package use as `Reusable Box`

Go to pick card and add picking orders to a batch (refer to the tutorial for batch)

Go to one of the picking that is added to a batch then click the `Batch` smart button

Or go to Operations -> batch transfer

Set a cluster package in the destination package column

validate batch

in the delivery order picking, click the hamburger icon to see that the cluster package is now in the source package column. Click `Put in Pack`

validate the picking

check the cluster package to see if the product is now removed

click `Package Transfer` smart button to see all of the pickings that used the cluster package
