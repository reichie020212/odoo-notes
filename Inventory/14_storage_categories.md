## Storage Categories

Enable `storage locations`

Go to configuration -> Storage Categories then create or update a record

Click `Locations` smart button to see the locations that uses the storage categories

Go to `Locations` form to set the storage category of a location

Go to `Putaway Rules` menu to set sublocation and storage category

There should be atleast two `putaway rules` incase the first location is already full

Please note that the sequence is necessary

Go to PO stock pickings and click the hamburger menu to see if putaway rules with sublocation takes place
