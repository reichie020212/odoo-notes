## Dispatch Management System

Enable `Batch, Wave and Cluster Transfer`, `Dispatch Management System`, `Storage location`, and `Multi-step Routes`

In warehouse, set outgoing shipment to two or three steps

in location, create a new location under the `output` locaion then check the `is a dock location` field

to configure vehicle category, go to `Fleet` app then config -> categories

to change the unit of measurements, go to general settings -> Units of measurement, then set the weight and volume

go to fleet form to configure a vehicle then set its category

in the inventory overview menum go to pick card then change the destination location to the dock location

validate pickings

go to `deliver orders` card, select some or all stock pickings, click Actions -> Add to batch. a pop-up will show

Fill-up necessary fields then click confirm

a batch will be created and will automatically redirected to it

set vehicle field, this will automatically compute the total weight/volume percentage

go to transfer tab to add/remove a stock picking

click `Map` header button to view the map to show all customer addresses

explore more features of map
