Go to a product form then `on hand` smart button

Update quantity by adding a new record or update the `Counted quantity` column then click apply to the updated row or click the `apply all` header button

To check the moves history of a product, go to products form then `In, Out` smart button

To do more inventory adjustments, go to Inventory -> Operations -> Physical Inventory

You can set selected products to zero or request a count or set to quantity on hand

click `History` to check moves history per product

To update `Cyclic counting` per location, go to configuration -> location -> then update `inventory frequency` field.

When an adjustment is done, the next adjustment will be based on the inventory frequency
